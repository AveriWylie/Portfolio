Computer Science student at Wilfrid Laurier University, with concentration in Big Data Systems. I build things I find interesting.

## Projects

### [AMP - Agentic Minecraft Player](https://github.com/AveriWylie/AMP)
Python-based Minecraft bot with guided and fully autonomous execution modes. Connects over raw TCP, decodes the binary protocol from scratch, parses live chunk data, pathfinds with A*, and uses Claude as an AI planning layer to translate natural language into in-game actions.

### [SHAYVERI](https://github.com/shahyaranfaz/SHAYVERI)
UCI-compliant chess engine in C++ achieving 2750 Elo on Lichess. Implements iterative deepening, null move pruning, late move reductions, singular extensions, bitboard move generation with magic number attack tables, and a multi-term evaluation function with all parameters tuned via SPSA and Texel tuning over self-play datasets.

### [mc-chunk-parser](https://github.com/AveriWylie/mc-chunk-parser)
Standalone version-aware Minecraft chunk parser in Python, written from scratch with no external dependencies. Existing Python libraries for chunk parsing are outdated or unmaintained — this fills that gap. Decodes binary chunk payloads into queryable block state data via NBT parsing, palette resolution, and heightmap extraction.

### [Space Invaders Recreation](https://github.com/AveriWylie/Space-Invaders-Recreation)
Space Invaders built in Python with Pygame. Includes state tracking, collision detection, timed power-ups, a local scoreboard.
